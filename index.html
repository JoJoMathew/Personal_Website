Delta Defence

A tower-defence game with a difference. What you’ll be tested on are your reactions and your co-ordination. Set to a 80s synthwave beat, you’ll battle your way through fifty levels of increasing difficulty.

In this game you’re required to match the colour of your tower to the colour of the enemy in order to be effective with your damage. Enemy waves will test your ability to have the correct towers placed at the right time. Towers can also be combined to form new colours for when the time calls. Each of the different level layouts will require thought about tactics and placement…

The full game also includes Survival mode where ever increasing odds will put you to the test and where your best scores can be pitted against the world with Game Centre leaderboard support.

Pi command pro


Pi Command gives you the ability to fully and effortlessly control your Raspberry Pi from your iOS Device.

Log in via SSH and view your Pi's Stats including:

• CPU Temperature
• Memory Used
• Disk Space
• Uptime
• External IP Address

Create custom buttons to enable you to easily run custom commands on your Raspberry Pi via SSH - iPhone 6S and iPhone 6S Plus users can also call these actions from the Home screen via 3D Touch Quick Actions.

Control your Pi's GPIO pins using easy toggle buttons and switches

• Toggle between Pin numbering or GPIO numbering
• Set pins as Outputs or Inputs
• Toggle pins true or false

Pi Command also provides a full Colour "Terminal" which you can use to run any command on your Raspberry Pi.

• Connects using SSH credentials
• Run commands
• View files or folders
